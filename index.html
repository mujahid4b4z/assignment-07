JAVASCRIPT (JS)

  • JavaScript is the world's most popular programming language.
  • JavaScript was intially created to "make web pages alive".
  • JavaScript is a lightweight,
    interpreted programming language.
  • JavaScript is dynamically-typed language, i.e.
    it performs type checking at runtime. This means
    that script written in dynamically-typed languages
    can compile even if they contain errors that will
    prevent the script from running properly (if at all).
  • It is full-fledged programming language that
    enables dynamic interactivity on websites when
    applied to an HTML document.
